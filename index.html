Este site é um guia didático sobre fundamentos de desenvolvimento web: HTML, CSS e JavaScript — com boas práticas, tecnologias relacionadas e um fluxo de projeto do início ao deploy.

O que é HTML

HTML (HyperText Markup Language) fornece a estrutura do conteúdo: títulos, parágrafos, listas, links e imagens. Use tags semânticas para melhorar acessibilidade e SEO.

Exemplo básico

<!doctype html>
<html lang="pt-BR">
  <head><meta charset="utf-8"></head>
  <body>
    <header><h1>Título</h1></header>
  </body>
</html>

O que é CSS

CSS (Cascading Style Sheets) controla o visual: cores, fontes, espaçamentos e layout. Prefira arquivos externos para separar apresentação e conteúdo.

Tipos de inclusão

  • Inline: dentro do atributo style (evite)
  • Internal: dentro de um <style>
  • External: arquivo style.css (recomendado)

Exemplo prático

body { font-family: system-ui, -apple-system, 'Segoe UI', Roboto, sans-serif; }
.container { max-width: 900px; margin: 0 auto; padding: 1rem; }

O que é JavaScript

JavaScript adiciona interatividade e comportamentos à página. Use ES6+ (let/const, arrow functions, template literals) para código mais legível.

Boas Práticas Profissionais

  • Acessibilidade (a11y): sempre usar alt em imagens, rótulos, contraste adequado e navegação por teclado.
  • Semântica: usar tags como <header>, <nav>, <main>, <section> e <footer>.
  • Responsividade: design mobile-first, Flexbox e media queries.
  • Organização: separar HTML/CSS/JS e usar nomes de classes claros.
  • Versionamento: explicar Git e GitHub (conceito, commits, branches).

Panorama das Tecnologias Web

Back-End

CI/CD, Docker, GitHub Actions e tipos de testes (unitário, integração, funcional).

Fluxograma de um Projeto Web

Etapas: ideia → planejamento → desenvolvimento → integração → testes → deploy → manutenção.
